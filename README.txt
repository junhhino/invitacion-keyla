# Invitación para Keyla

Esta versión incluye:
- Conversación animada con 4 escenas.
- Woody y Buzz apareciendo por separado.
- Fondos blancos removidos o suavizados desde las imágenes.
- Botón de WhatsApp configurado con +595 972 682 633.
- Invitación para Toy Story 5 en Cinemark Paseo La Galería.

## Cómo publicar gratis en GitHub Pages

1. Crea un repositorio en GitHub llamado `invitacion-cine-keyla`.
2. Sube `index.html` y la carpeta `assets`.
3. Entra a Settings -> Pages.
4. En Branch selecciona `main` y `/root`.
5. Copia el enlace que GitHub te entregue y envíaselo a Keyla.
